# 🌙 A Quieter Path to Sleep

A beautiful, interactive sleep guide web app to help you build better bedtime habits, calculate your ideal sleep schedule, and wind down with a guided breathing exercise.

**Live site:** [your-username.github.io/sleep-guide](https://your-username.github.io/sleep-guide)

---

## Features

- **Sleep Schedule Calculator** — Set your wake time in 15-minute increments and instantly get your ideal wind-down and lights-out times
- **Evening Wind-Down Checklist** — 7 science-backed habits to check off each night, filterable by category (Schedule, Environment, Racing Mind)
- **4–7–8 Breathing Exercise** — Animated guided breathing to quiet a racing mind before sleep

---

## How to Use

Just open `index.html` in any browser — no installation or build step required.

---

## Tech Stack

- [React 18](https://react.dev/) via CDN
- [Babel Standalone](https://babeljs.io/docs/babel-standalone) for in-browser JSX
- Vanilla CSS — no frameworks

---

## Deployment

Hosted with [GitHub Pages](https://pages.github.com/). To deploy your own copy:

1. Fork or clone this repo
2. Go to **Settings → Pages**
3. Set source branch to `main`, folder to `/ (root)`
4. Save — your site will be live in ~1 minute
